A propos

Actuellement étudiante en informatique, mon objectif est de participer à la réalisation de plusieurs projets dans le domaine du développement informatique afin d'appliquer mes connaissances et d'acquérir de nouvelles compétences dans ce domaine.

HTML est un langage informatique utilisé sur l'internet pour créer des pages web. HyperText Markup Language traduit littéralement en français "langage de balisage d'hypertexte" porte bien son nom puisqu'effectivement ce langage permet de réaliser de l'hypertexte à base d'une structure de balisage.

Le terme CSS est l'acronyme anglais de Cascading Style Sheets qui peut se traduire par "feuilles de style en cascade". Le CSS est un langage informatique utilisé sur l'internet pour mettre en forme les fichiers HTML ou XML. Ainsi, les feuilles de style, aussi appelées les fichiers CSS, comprennent du code qui permet de gérer le design d'une page en HTML.

Le Javascript est un autre langage informatique utilisé sur les pages web. Il a la particularité de s'activer sur le poste client, en d'autres termes, l’ordinateur recevra le code et l'exécutera, contrairement à d’autres langages qui sont exécutés du côté serveur. L'interprétation du code est effectuée par un navigateur internet tel que Firefox ou Google Chrome.
